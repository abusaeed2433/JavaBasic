## Using Named Groups
- group can be named instead of using with number,
- Format:
  ```
  (?<groupName>pattern)
  ```
- The group name must consist of
  - Only `letters` and `digits`,
  - `a to z`, `A to Z`, and `0 to 9`,
  - Must start with `letter`,
